# VSCODE: Visual Sub-COnsciousness DEscriber

프로젝트 구동하는 법

- NodeJS 설치(되어 있으면 패스)
- 프로젝트 경로의 터미널에서 아래 스크립트 돌리기

```bash
npm install
npm run dev
```
